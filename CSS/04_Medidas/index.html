Unidades de medidas en Css

En Css al expresar el valor de una medida debemos poner siempre el numero(cantidad) seguido del indicador de la unidad de medida

Tipos de Unidades: Absolutas y Relativas

Unidades absolutas

Las unidades absolutas son un tipo de medida fija que no cambia, que no depende de otro factor(sin medidas fijas)

mm: milímetros

cm:centimetros

cm:centimetros

in:pulgada

pt:puntos

pc:picas

px: pixel

Unidades relativas

Las unidades relativas son mas flexibles, es decir, se adaptan mas facilmente a los diferetes medios

Su valor se calcula siempre en base a otro elemento
